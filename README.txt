Prerequisites:
	1. Python
	2. Django

1. Download the project using the GitHub repo URL
2. In the terminal,
	-->change the current directory to the folder project using the cd command
	-->to run the server execute the following command: python manage.py runserver
	-->The above command will generate a URL http://127.0.0.1:8000/ in the command prompt.
3. Paste the URL in the web browser to get the result.
